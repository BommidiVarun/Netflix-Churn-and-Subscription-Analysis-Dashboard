# Netflix-Churn-and-Subscription-Analysis-Dashboard


# Project Overview
This Power BI dashboard analyzes Netflix customer churn, subscription revenue, watch hours, and user behavior across regions, genres, and devices.  
The goal is to identify churn patterns and revenue-driving segments.

# Key KPIs
- Total Watch Hours: 58.25K
- Total Monthly Revenue: 68.42K
- Total Churned Customers
- Revenue by Subscription Type
- Watch Hours by Region, Genre & Device

# Tools & Skills Used
- Power BI
- Data Cleaning
- Data Modeling
- Business Insights

# Key Insights
- Premium subscription generates the highest revenue
- South America and Europe show higher churn levels
- Mobile and TV are the most used devices
- Drama and Romance genres have the highest watch hours

# Business Value
This analysis helps streaming platforms:
- Reduce customer churn
- Improve subscription pricing strategy
- Target high-risk regions and user segments
